Media types

Timeline supports many ways to illustrate your story

Just upload files

You may already have images, video and audio you want to use. Timeline NG lets you upload those files right in the editor. You have to be signed in with an atmosphere account to upload files. If you are, they'll be saved to your own server.

Paste a URL, get an embed

If the media you want to include is already on the web, use the editor's URL / Embed tab, or the media.url field if you're writing JSON directly. Timeline NG looks at the shape of the URL itself and picks the right way to show it. There's nothing special to configure: paste the same link you'd copy from your browser's address bar or a site's "Share" button, and the player figures out the rest.

If a URL doesn't match any of the patterns below, Timeline NG tries it as a generic embedded frame as a last resort. Some sites block this (via X-Frame-Options), in which case the slide will show a blank space or refuse to load — if that happens, look for that site's own "embed code" option (see Raw HTML below) or link out to it directly instead.

Recognized URL patterns

Paste a link like…You get
A direct file link ending in .jpg, .png, .gif, .svg, or .webpThe image itself
A direct file link ending in .mp3, .wav, .m4a, .ogg, or .flacAn audio player
A direct file link ending in .mp4, .webm, .mov, or .ogvA video player
A direct link ending in .pdfAn inline PDF viewer
A YouTube link (youtube.com/watch?v=… or youtu.be/…)A video player. Start/end time parameters (?t=, &start=, &end=) are honored.
A Vimeo linkA video player
A DailyMotion or Wistia linkA video player
A SoundCloud track linkAn audio player
A Spotify track, album, or playlist link (or a spotify: URI)An audio player
A Bluesky post link (bsky.app or a compatible app like Blacksky or Mu.social)The post, embedded
A TikTok video linkThe video, embedded
A Twitter/X status linkThe post, embedded
A Wikipedia article linkA summary card with thumbnail and excerpt
A Wikimedia Commons file linkThe full image
A Google Maps linkAn embedded map
A Google Drive or Google Docs/Sheets/Slides linkAn embedded document viewer
A Flickr photo page linkThe image itself

Raw HTML

If a site gives you an "embed code" snippet rather than a plain link (often starting with <blockquote> or <iframe>), you can paste that whole snippet into the media field instead of a URL. Timeline NG detects that it's markup, not a link, and renders it as-is. Note: embeds that require script tags will not work—those get stripped out for safety reasons.

Atmosphere Apps

Most people start with Bluesky, but there are other atmosphere apps coming out all the time. Once you're signed in here, we can help you bring data from your other apps into your timeline. Use the "atmosphere" tab in the media editor to add them, and let us know if there are other apps you think we should support.
